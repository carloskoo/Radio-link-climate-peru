# Figures

This directory contains the figures generated for the cross-dataset consistency analysis between ERA5/ERA5-Land and NASA POWER.

## consistency_combined_ieee.png

Combined consistency figure comparing ERA5/ERA5-Land and NASA POWER for relative humidity and precipitation across representative radio links in Peru:

- L09-A: Coastal region
- L03-A: Andean region
- L10-A: Rainforest region

The dashed black line represents the ideal 1:1 agreement, while the red line represents the linear regression fit.

## mean_correlation.png

Mean Pearson correlation coefficient by variable:

- SP: Surface pressure
- RH: Relative humidity
- PRECIP: Precipitation
- T: Temperature

## mean_rmse.png

Mean RMSE by variable after unit harmonization and bias correction.
