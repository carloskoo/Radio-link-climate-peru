# Radio-link-climate-peru

This repository contains the reproducible materials associated with the thesis:

**Evaluation of the influence of climatological factors on point-to-point radio link attenuation in regions of Peru.**

The repository is organized into two main modules:

1. `consistency-github-base/`  
   Cross-dataset consistency analysis between ERA5/ERA5-Land and NASA POWER.

2. `attenuation-github-base/`  
   Atmospheric attenuation modeling using ITU-R P.676 and ITU-R P.838.
