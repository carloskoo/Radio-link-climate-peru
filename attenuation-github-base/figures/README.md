# Figures

This directory contains the figures generated during the attenuation analysis workflow.

## Figures

### fig_gaseous_attenuation_q1.png
Regional comparison of gaseous attenuation computed using ITU-R P.676.

### fig_rain_attenuation_q1.png
Regional comparison of rain attenuation computed using ITU-R P.838.

### fig_total_attenuation_q1.png
Total atmospheric attenuation obtained by combining gaseous and rain attenuation.

### fig_rain_dominance_ratio_q1.png
Relative contribution of rain attenuation to total attenuation.

### fig_comparative_mechanisms_q1.png
Comparative summary of attenuation mechanisms across representative radio links.
